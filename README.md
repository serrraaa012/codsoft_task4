## Quizify - Android Quiz App
A feature-rich, interactive quiz application built with Java for Android that tests your knowledge across multiple categories with an intuitive and visually appealing interface.

## Features
Multiple Quiz Categories

Luxury and Supercars (10 questions)

Formula 1 Racing (10 questions)

Car Brands and Logos (5 questions)

Interactive Quiz Experience

One question at a time with multiple-choice options

Immediate feedback on answers (Correct/Incorrect)

Score tracking throughout the quiz

Visual indication of correct answers with color highlighting

Progress tracking showing current question number

Modern UI/UX

Custom gradient backgrounds with smooth transitions

Responsive design for all screen sizes

Custom radio button styling with visual feedback

Beautiful result screen with motivational messages

Card-based quiz selection interface

Circular score display with animations

Additional Features

Random quiz selection

Retry quiz option

Score persistence across sessions

Custom font support

Image background support

Visual feedback for answer selection

## Tech Stack
Language: Java

Minimum SDK: API 24 (Android 7.0 Nougat)

Architecture: MVC (Model-View-Controller)

UI Components:

Material Design Components

LinearLayout and ConstraintLayout

RecyclerView for quiz list

RadioGroup/RadioButton for options

CardView for quiz items

Libraries Used
AndroidX AppCompat

Material Design Components

RecyclerView

CardView
